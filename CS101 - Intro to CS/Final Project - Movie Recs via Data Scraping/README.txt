This program requires the user to install BeautifulSoup.

A  guide on installing this library can be found at :
https://www.crummy.com/software/BeautifulSoup/bs4/doc/

In general, the installation is very straightforward.

It is important to note that the inputs to the program are case sensitive, such that names are capitalized as well as genres, etc.

Finally, if the user intentionally inputs bad data, the program will crash on some of the narrowing functions, not the menu though.
